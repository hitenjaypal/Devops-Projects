# What You Will Learn in This Project

## Terraform Foundations
- Provider setup and version constraints
- Variables and reusable infrastructure code
- Outputs and how to expose useful IDs
- Conditional resource creation using `count`

## AWS Networking Basics
- VPC and CIDR planning
- Subnet and route table relationship
- Internet Gateway role
- Security group defaults and safe access posture

## Infrastructure Workflow
- `init` -> `plan` -> `apply` -> `destroy`
- Reading execution plans before changes
- Understanding Terraform state and drift

## Cost-Aware Engineering
- Building minimal architecture first
- Enabling optional resources only when needed
- Practicing clean teardown to avoid charges

## Interview Readiness
- Terraform and VPC conceptual Q&A
- Real debugging scenarios and production-style answers
